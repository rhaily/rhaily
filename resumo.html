Resumo HTML

Um documento HTML válido precisa seguir obrigatóriamente a estrutura composta pelas tags html,head e body. A instrução !Doctype. Esta estrutura está informada em uma documentação que descreve todos os detalhes do Html,no caso as tags e atributos,e como os navegadores devem considerar e interpretar estas tags,esta documentação é chamada de "especificação do HTML". O HTML permite adicionar texto á página,incluindo cabeçalhos,paragráfos,tintas e outros elementos de texto. Como por exemplo: CSS (Cascading Style sheets) que serve para descrever a apresentação de documentos HTML e XML na web. O CSS permite separar a estrutura do conteúdo HTML de apresentação visual, permite que os designers controlem a aparência da página da web sem afetar o seu conteúdo. Com o CSS, é possível definir estilos como cor de fundo, a fonte, tamanho, espaçamento, posicionamento e outros aspectos de design.

Em resumo, as cores no CSS podem ser especificadas pelo nome, código RGB, código HEX ou palavra-chave e são usadas como valores de propriedades de estilos para controlar a cor de elementos HTML. As cores no CSS podem ser epecifícadas de várias maneiras, incluindo; Nome. Como por exemplo: Red, Blue, Green.

As imagens podem ser manipulados no HTML usando a tag "img". A tag "img" permite especificar a URL da imagem a ser exibida, bem como outras propriedades opcionais, como largura, altura, descrição da imagem, entre outros. O posicionamento de elementos em HTML é uma questão importante para a organização e layout de uma página web. Através do uso da técnicas de posicionamento, é possível definir a disposição dos elementos. Uma das técnicas mais comuns de posicionamento é o uso de CSS, que permite definir as proprieddes de posicionamento dos elementos. Com CSS, é posível definir o posicionamento absoluto de um elemento em relação a um ponto específico na tela, ou ainda definir o posicionamento relativo em relação a outros elementos na página
